This is a fashion website designed for LUKU LEGIT, focused on showcasing stylish thrifted footwear and original products with a clean, luxurious user experience.
It contains a responsive modern UI, Mobile friendly design, Stylish product showcase, and a clean layout.
Goal was to create a visually attractive and user friendly website that reflects the energy, authenticity and modern streetwear vibe of LUKU LEGIT.
Built with HTML5, CSS, Javascript and works smoothly on mobile phones, tablets and desktop devices.
To install, clone git https://github.com/jimmyCpu01/LUKU-LEGIT.git, open project folder and run index.html on your browser.
